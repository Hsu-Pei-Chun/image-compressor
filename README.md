# Image Compressor

A simple browser-based image compression tool. Compress PNG, JPG, and WebP images to a target file size.

## Demo

https://hsu-pei-chun.github.io/image-compressor/

## Features

- Support PNG, JPG, WebP formats
- Configurable target file size (KB)
- Multiple output formats (JPEG, WebP, PNG)
- Drag and drop upload
- Batch processing
- 100% client-side (no server upload)

## Usage

1. Set target size (default: 200 KB)
2. Choose output format
3. Drag and drop images or click to upload
4. Download compressed images

## Tech Stack

- HTML5
- CSS3
- Vanilla JavaScript
- Canvas API
